# S03E06 全职做6年开源后的忠告

## 背景

倘若你查阅一下 [Linux Distribution Timeline](https://github.com/FabioLolix/LinuxTimeline/releases/) 便会发现，[Debian](https://www.debian.org)  作为“一个完全自由的操作系统！”几乎是高山一般的存在。放眼全球基于 Debian 的发行版多如牛毛，但基于 Debian 的国产 Linux 发行版却屈指可数，而[铜豌豆 Linux](https://www.atzlinux.com/) 正是其中之一。

2025年11月19日，[开源面对面](https://osf2f.net/)邀请了铜豌豆 Linux 创始人肖盛文一起谈了谈关于 Debian 和全职开源的话题。

1. 为什么会选择基于 Debian 进行开发，维护一个发行版都需要做哪些工作？
2. 铜豌豆的用户画像是什么，社区建设如何？
3. 回顾一下参加过的 Debian 年会，以及参会分享。
4. 介绍下国内 Debian 贡献者的现状。
5. 关于[星际译王的问题](https://lwn.net/Articles/1032732/)。
6. 谈谈全职做开源的感受和经验分享。
